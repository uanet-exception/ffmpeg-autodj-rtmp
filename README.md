# **ffmpeg-autojd-rtmp**

## **Description**
`ffmpeg-autojd-rtmp` - shell script that invokes ffmpeg process to stream mp3 files to rtmp server.

## **Dependencies**
* `mbuffer`
* `ffmpeg`
* `jq`

## **Usage**
Copy some mp3s into music directory, set RTMP server address and run `./stream.sh` script.
